# CampusConnect 🎓

CampusConnect is a unified digital hub designed for higher education institutions to manage extracurricular activities, hackathons, workshops, and cultural celebrations. The system resolves friction in student engagement and extracurricular administration through automated QR ticketing, AI-powered team matching, and gamified achievement progression.

---

## 🚀 Key Features

### 1. Automated QR-Based Ticketing 🎟️
- **Dynamic Ticket Generation:** Replaces paper attendance sheets and physical ticketing with secure, dynamic, scanner-verifiable QR tickets.
- **Unique Ticket Format:** Generates unique ticket tokens in the format `CC-REG-{uuid}`.
- **Easy Registration:** Registers contact data (phone, field of study, year, college, transaction ID for paid events) and tracks seat capacities.

### 2. Smart Attendance Verification & Check-in ⏱️
- **Instant Attendance Logging:** Simulating or scanning a ticket check-in logs the exact timestamp (`attendedAt`) in the database.
- **Points Allocation:** Students earn **+50 XP** for each event checked into, which updates their national and college leaderboard ranks.
- **Audit Logs:** Logs system-wide check-in logs in a dedicated dashboard feed for audit trails.

### 3. Gamified Achievement Progression 🏆
- **Unlockable Badges:** Accumulating event check-ins dynamically awards badges:
  - **1 Event Attended:** *Volunteer* Badge
  - **2 Events Attended:** *Collaborator* Badge
  - **5 Events Attended:** *Tech Enthusiast* Badge
- **Leaderboards:** Real-time leaderboard rankings (national and college level) based on XP accumulation.
- **Verifiable Certificates:** Generates beautiful certificates for accomplishments which can be downloaded as PDF.

### 4. AI-Powered Skill-Based Team Matching 🧠
- **Dynamic Compatibility Scoring:** Computes compatibility percentages between students and project teams based on team skill requirements (e.g. React, Python, UI/UX, ML).
- **Match Matrix:** Starts at a base compatibility score of 50% and increments dynamically (e.g., +10% per matching skill) up to a maximum of 98%.
- **Teammate Invites:** Panel to send, receive, accept, or decline team membership invites.

### 5. Multi-Theme Styling Engine 🎨
- **Vanilla CSS Engine:** Uses CSS custom variables (`globals.css`) for high performance and clean structure.
- **Three Core Themes:** Toggles smoothly between **Light**, **Dark**, and **High Contrast** (accessibility-friendly) themes.
- **State Persistence:** Custom preferences (theme, notification configs) are saved and applied dynamically by altering the `data-theme` attribute on the root HTML document.

### 6. Client-Side Certificate Exporter 📄
- **PDF Generation:** Converts digital credentials to downloadable PDFs using `html2canvas` and `jspdf` directly on the client side.

### 7. AI-Generated Achievement Certificate 🖼️
-*Dynamic Unlock:** The premium certificate is automatically generated by the system and becomes unlocked and downloadable as a PDF once a student successfully completes the event

## 🗄️ Database Schema Overview (Prisma)

The application database schema consists of **13 key models**:

```
                +-----------------+
                |      User       | <-------------------+
                +--------+--------+                     |
                         |                              |
         +---------------+---------------+              |
         |               |               |              |
         v               v               v              v
  +--+-------+     +--+-------+     +--+-------+  +--+-------+
  |UserSkill |     |UserBadge |     |Certificate  |Registration
  +----------+     +----------+     +----------+  +----------+
         ^               ^                              ^
         |               |                              |
         v               v                              |
  +--+-------+     +--+-------+                         |
  |  Skill   |     |  Badge   |                         |
  +----------+     +----------+                         |
                                                        |
                                                        v
                                                 +--+-------+
                                                 |  Event   |
                                                 +--+-------+
                                                        ^
                                                        |
                                                        v
                                                 +--+-------+
                                                 |  Team    |
                                                 +--+-------+
                                                        ^
                                                        |
                                                        v
                                                 +--+-------+
                                                 |TeamMember|
                                                 +----------+
```

- **`User`**: Profiles, points tallies, national/college ranks, settings.
- **`Skill` & `UserSkill`**: Skill catalogs and student competencies (1-100).
- **`Badge` & `UserBadge`**: Unlockable milestones.
- **`Certificate`**: Earned credentials linked to verification URLs.
- **`Event`**: Details about title, timings, venue, pricing, categories, and capacities.
- **`Registration`**: Links users to registered events, keeps check-in status and QR strings.
- **`Team`**, **`TeamMember`**, & **`TeamInvitation`**: Manages hackathon project teams and invites.
- **`Activity`**: Audit logging for student activities and feed.
- **`SupportMessage`**: In-app chat messages with the college support desk.

## 💻 Local Installation & Setup

Follow these steps to run the project locally on your machine:

### Prerequisites
- Node.js (v18.x or later recommended)
- npm (v9.x or later)
- PostgreSQL database instance (Neon Serverless PostgreSQL recommended)

### Step 1: Clone & Access Directory
Navigate to the `campusconnect` folder in your workspace:
```bash
cd campusconnect
```

### Step 2: Install Node Modules
Fetch all project dependencies:
```bash
npm install
```

### Step 3: Setup Environment Variables
Create a `.env` file in the root of the `campusconnect` folder and define your database connection string:
```env
DATABASE_URL="postgresql://username:password@hostname:5432/db?sslmode=require"
```
*(You can reference the configuration template in [.env.example](file:///c:/Users/ANANDHU%20A/campus%20connect/campusconnect/.env.example))*

### Step 4: Push the Database Schema
Apply the Prisma schema to your target PostgreSQL database and generate the Prisma Client:
```bash
npx prisma db push
```

### Step 5: Seed Initial Data
Seed essential data (skills dictionary, default badges, and a test profile):
```bash
npx prisma db seed
```

### Step 6: Run Development Server
Start the Next.js local server:
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) in your web browser.

## 🧪 Testing & Verification Protocol

Verify the features of the application locally using this structured walkthrough:

### 1. Log In to Test Account
- Click **Get Started** on the landing page.
- Log in with the preloaded test email: `alex.chen@campusconnect.edu`.
- Verify the system loads your user data, XP points, and welcome activity feed.

### 2. Verify Theme Customization
- Go to **Settings** in the application menu.
- Toggle between **Light**, **Dark**, and **High Contrast** themes.
- Confirm that the UI colors adjust instantaneously. The engine modifies the `data-theme` attribute on the `<html>` document, instantly updating global HSL CSS variables.

### 3. Register for an Event
- Go to the **Events** section.
- Browse the available events and click **Register** on a workshop or hackathon.
- Fill out the form, complete the simulation, and verify that a unique QR Ticket is rendered with a ticket ID prefix of `CC-REG-`.

### 4. Test Smart Check-in Simulation
- When logged in, trigger or scan the attendance endpoint for your event registration (calls `POST /api/events/[id]/check-in`).
- Verify that:
  - Your registration status updates to **Checked In** with a timestamp (`attendedAt`).
  - Your user profile gains **+50 XP** points.
  - The milestone badge awards are updated. Check the **Achievements** tab to confirm that milestones (Volunteer, Collaborator, or Tech Enthusiast) are unlocked as you accumulate attendances.

### 5. Test AI Team Matching
- Go to the **Teammate Finder / Team Builder** panel.
- Select a team (e.g. *Team Nexus*) to see your compatibility score.
- Verify that the score starts at 50% and scales up with each matching skill you possess (React, Python, etc.).
- Try sending a team invite or checking received invitations.

### 6. Export Credentials to PDF
- Navigate to the **Certificates & Achievements** page.
- Choose a certificate and click **Download PDF**.
- Verify that a clean, formatted PDF of the certificate is generated and saved locally on your device.
